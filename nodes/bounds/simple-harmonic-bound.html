Simple bound on harmonic sum

Dependencies:

  1. Integration bound

Let $H(n) = \sum_{i=1}^n \frac{1}{i}$. Then $\ln n + \frac{1}{n} \le H(n) \le \ln n + 1$.

This is a direct application of the integration bound.
