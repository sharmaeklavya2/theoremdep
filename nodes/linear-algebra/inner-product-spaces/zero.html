Zero in inner product

Dependencies:

  • Inner product space

$\langle 0, v \rangle = \langle v, 0 \rangle = 0$.

Proof

\[ \langle 0, v \rangle = \langle 0 + 0, v \rangle = \langle 0, v \rangle + \langle 0, v \rangle \Rightarrow \langle 0, v \rangle = 0 \]

\[ \langle v, 0 \rangle = \langle v, 0 + 0 \rangle = \langle v, 0 \rangle + \langle v, 0 \rangle \Rightarrow \langle v, 0 \rangle = 0 \]
