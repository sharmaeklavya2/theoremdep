Homogeneous linear equations with more variables than unknowns

Dependencies:

  • Rank of a homogenous system of linear equations

Let $AX = 0$ be a system of $m$ linear equations in $n$ variables where $m < n$. Then there is a non-trivial solution.

Proof

$\operatorname{rank}(A) \le m < n \Rightarrow$ there is a non-trivial solution.

Dependency for:

  • Linearly independent set is smaller than a span
